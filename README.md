# myAddress
連絡先を登録・表示するWEBアプリケーション

### 言語
JavaScript

### フレームワーク
vue, vue-router, vuex, vuetify 

### API
firebase

### DBライブラリ
Cloud Firestore


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
